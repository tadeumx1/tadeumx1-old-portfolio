  • ETESP APP

    MOBILE

  • C# MAT File Reader

    WEB

    O C# MAT File Reader é um projeto que venho desenvolvendo, que é um leitor de arquivos .Mat que geralmente são gerados no MatLab usando, a biblioteca CSMATIO que foi feita pelo Sérgio Ferreira(AlienEngineer) e está disponível no seu GitHub (https://github.com/AlienEngineer/CSMatIO) com ela posso descompactar os arquivos .Mat e usando C# e ASP estou fazendo, o processo de leitura desses arquivos .Mat considerando, todas as variáveis dos arquivos e também todos os tipos de variáveis do MatLab incluindo as Structs estou desenvolvendo esse projeto, para o TPN (Tanque de Provas Númerico da USP) e essa aplicação terá um importante uso no sistema que eu e minha equipe estamos desenvolvendo. Por questões corporativas ainda não posso disponiblizar o código fonte e a demo desse projeto.
